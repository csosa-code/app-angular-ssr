Acerca de PokeApp

¿Qué es PokeApp?

PokeApp es una aplicación web para explorar el mundo Pokémon. Permite consultar la lista de Pokémon disponibles y ver el detalle completo de cada uno: estadísticas, tipos, habilidades, altura, peso y más.

Funcionalidades

  • Lista de Pokémon — Navega por el catálogo con paginación
  • Vista detalle — Información completa de cada Pokémon (stats, tipos, habilidades)
  • Datos en tiempo real — Consume la API pública de PokeAPI

Tecnología

La aplicación está desarrollada con Angular y utiliza Server-Side Rendering (SSR). Esto significa que las páginas se generan en el servidor antes de enviarse al navegador, lo que mejora el SEO, la velocidad de carga inicial y la experiencia en dispositivos con conexión limitada.
